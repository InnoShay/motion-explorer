## 🌟 Overview
**Motion Explorer** is not just an educational web application—it's a futuristic learning laboratory. It takes the original NCERT text and breathes life into it through high-fidelity CSS animations, real-time physics simulations, and a deeply integrated AI tutor. 

The goal? To help students truly *understand* motion by seeing, interacting, and experimenting, rather than simply memorizing definitions.

---

## ✨ Key Features

### 🛸 Immersive Boot Loader
A cinematic, physics-inspired launch sequence featuring a 3D atom animation, deep-space parallax grid, and dynamic loading progress, setting a premium "EdTech" tone from the very first second.

### 🎮 Interactive Simulations
- **Reference Points:** Drag and drop elements to understand relative motion.
- **Distance vs. Displacement:** Compare direct paths vs. meandering trails in real-time.
- **Speed & Velocity:** Interactive speedometers and vector arrows.
- **Graphing Engine:** Watch distance-time graphs draw themselves dynamically as virtual cars move across the screen.

### 🤖 NOVA: Contextual AI Tutor
A fully custom, responsive AI tutor integrated directly into the learning flow. NOVA provides real-time feedback, celebratory animations, and helpful hints without ever giving away the direct answers.

### 💎 Ultra-Modern UI/UX
Built with a cutting-edge aesthetic using **Glassmorphism**, vibrant cyan and purple gradients, dynamic hover effects, and buttery-smooth 60FPS CSS transitions.

## 🏃‍♂️ Run Locally

Running the project on your own machine is incredibly simple. No build steps required.

1. **Clone the repository**
   ```bash
   git clone https://github.com/InnoShay/motion-explorer.git
   ```
2. **Navigate to the directory**
   ```bash
   cd motion-explorer
   ```
3. **Launch a local server**
   ```bash
   npx serve .
   ```
   *(Or simply open `index.html` in your favorite modern browser!)*
